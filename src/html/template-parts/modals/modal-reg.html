Правила аукциона
1. До начала
  • a. Каждый участник должен заплатить фиксированное количество токенов для участия в торгах за лот.
  • b. Пользователь может отменить участие в аукционе и вернуть токены до момента начала торгов. Цена за вход определяется при создании лота.
  • c. Если до начала торгов не соберется определенное количество участников, то торги автоматически отменяются.
2.Процесс торгов
  • a. Процесс торгов начинается с ожидания первого повышения цены от участников. Цена лота начинается с цены в 0 токенов.
  • b. Повышение цены фиксированное и устанавливается при создании лота. Участник не может перебить текущую цену, если новая цена больше его текущего баланса.
  • c. После каждого повышения цены. Есть фиксированное количество времени (например 3 мин) для дальнейшего повышения цены другими участниками.
  • d. Если никто не перебил цену за указанное время, то победителем торгов объявляется последний поднявший цену участник.
